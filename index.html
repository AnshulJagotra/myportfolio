E-Commerce Site

The site features a user-friendly interface built with React, Tailwind CSS, and Material-UI, allowing for efficient product browsing, filtering, and detailed view. Users can manage their shopping carts and checkout securely with Razorpay integration. The platform supports secure user authentication through JWT, ensuring safe access and transactions. On the backend, Spring Boot and MySQL handle scalable server-side operations and RESTful APIs. An admin panel enables efficient management of product listings, user accounts, and orders. This project aims to deliver a robust, secure, and engaging shopping experience, enhancing user satisfaction and operational efficiency.
